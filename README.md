SFCSFS
========

[SFC-SFS](https://gc.sfc.keio.ac.jp/)を快適にスクレイピングし、次世代のイノベーションを加速するためのRuby用ライブラリです。SFC-SFSクローラーの車輪の再発明が常時行われている問題を発見し、解決するために作成しています。

機能
-----

* My時間割の取得
* 次学期プランの取得・作成
* 課題の取得
* 残留届けページヘの移動
